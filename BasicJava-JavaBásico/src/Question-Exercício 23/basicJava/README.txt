* This is an example of how to use averaging in Eclipse.

Question Statement:

* Make a program to read an indeterminate number of data, each containing the age of an person. The last piece of data, which will not be included in the calculations, contains a negative age value. Calculate and print the average age of this group of persons. If a negative value is entered the first time, display the message "Unable to calculate".

Input:                / Input:
31                    / -20
27                    /
46                    / Output:
-5                    / Unable to calculate

Output:
34.67
