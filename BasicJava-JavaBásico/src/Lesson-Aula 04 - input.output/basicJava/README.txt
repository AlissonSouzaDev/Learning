(EN - US) Notes: 

* If you use 'println' to print the double for floating point numbers it will take the international localization and print the number with ""."". However, if you use 'printf' it will follow your computer's timer and print in PT-BR, i. e., with "","". To always print with ""."" just put 'Locale' before the Scanner.

(PT - BR) Observações:

* Se você utilizar o 'println' para imprimir o double nos casos de números com pontos flutuantes ele vai pegar a localização internacional e vai imprimir o número com ""."". Contudo, se você utilizar 'printf', ele vai seguir o temporizador do seu computador e vai imprimir em PT-BR, ou seja, com "","". Para imprimir sempre com ""."", é só colocar o 'Locale' antes do Scanner.
