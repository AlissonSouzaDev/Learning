(EN - US) Table Test:

* The table test is done to test the student's learning, through a finished code the student has to simulate the execution of this code as if it were executed in the IDE, writing the respective input and output values that the code should execute in the IDE.

- Question 01:

x = 3;
y = 0;

for (i = 0; i < x; i++) {
	
	System.out.print (i + ",");

	y = y + 5;

	System.out.println (y);

}

- Answer:

Values of the variables at each repetition of the code:

x:          / y:         / i:         / Output:
3           / 0          / 0          / 0,5
            / 5          / 1          / 1,10
            / 10         / 2          / 2,15
            / 15         / 3

- Question 02:

            / 9          / 3          / 4 9

(PT - BR) Teste de Mesa:

* O teste de mesa é feito para testar o aprendizado do aluno, através de um código finalizado o aluno tem que simular a execução deste código como se ele fosse executado na IDE, escrevendo os respectivos valores de entrada e saída que o código deveria executar na IDE.

- Questão 01:

x = 3;
y = 0;

for (i = 0; i < x; i++) {
	
	System.out.print (i + ",");

	y = y + 5;

	System.out.println (y);

}

- Resposta:

Valores das variáveis a cada repetição do código:

x:          / y:         / i:         / Output:
3           / 0          / 0          / 0,5
            / 5          / 1          / 1,10
            / 10         / 2          / 2,15
            / 15         / 3
