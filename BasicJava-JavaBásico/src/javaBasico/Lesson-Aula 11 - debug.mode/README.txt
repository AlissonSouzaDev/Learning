(EN - US) Notes:

* Lesson designed to test Eclipse's debug mode.

Debug mode commands:

Crtl + Shift + B = Toggle breakpoint;
Alt + Shift + D + J = Opens the debug mode window;
F6 = Advances one test in debug mode.

(PT - BR) Observações:

* Aula feita para testar o debug mode do Eclipse.

Comandos do debug mode:

Crtl + Shift + B = Toggle breakpoint;
Alt + Shift + D + J = Abre a janela do debug mode;
F6 = Avança um teste no debug mode.
