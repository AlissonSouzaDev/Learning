(EN - US) Question Statement:

* Write a program that repeats reading a password until it is valid. For each incorrect password, write the message "Password Invalid". When the password is correctly entered printed the message "Access Allowed" and the algorithm terminated. Consider that the correct password is the value 2002.

Input:
2200
1020
2022
2002

Output:
Password Invalid
Password Invalid
Password Invalid
Access Allowed

Quando a senha for informada corretamente deve ser impressa a mensagem "Access Allowed" e o algoritmo encerrado. Considere que a senha correta é o valor 2002.

Entrada:
2200
1020
2022
2002

Saída:

Password Invalid
