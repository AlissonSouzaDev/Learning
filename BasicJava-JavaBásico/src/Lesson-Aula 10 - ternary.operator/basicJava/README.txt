(EN - US) Notes:

* The structure of the example ternary operator corresponds to this code:

double price = 34.5;
double discount;
		
		if (price < 20) {
			
			discount = price * 0.1;
			
		} else {
			
			discount = price * 0.05;
			
		}
		
		System.out.println(discount);

* This ternary structure can replace the if else structure when you want to decide a value based on a condition.
