(EN - US) Notes:

* The structure of the switch case example corresponds to this code:

Scanner sc = new Scanner(System.in);
		
		int x = sc.nextInt();
		String day;
		
		if (x == 1) {
			
			day = "sunday";
			
		} else if (x == 2) {
			
			day = "monday";
			
		} else if (x == 3) {
			
			day = "tuesday";
			
		} else if (x == 4) {
			
			day = "wednesday";
			
		} else if (x == 5) {
			
			day = "thursday";
			
		} else if (x == 6) {
			
			day = "friday";
			
		} else if (x == 7) {
			
			day = "saturday";
			
		} else {
			
			day = "invalid value";
			
		}
		
		System.out.println("Day of the week: " + day);
		
		sc.close();
