(EN - US) Question Statement:

* Make a program that reads N real numbers and stores them in a vector. Then
- Print all the elements of the vector;
- Show on the screen the sum and the average of the elements of the array.

(PT - BR) Enunciado da Questão:

* Faça um programa que leia N números reais e armazene-os em um vetor. Em seguida:
- Imprimir todos os elementos do vetor;
- Mostrar na tela a soma e a média dos elementos do vetor.
