(EN - US) Question Statement:

* Make a program that reads a positive integer N (maximum = 10) and then N integers and stores them in an array. Then display on the screen all the negative numbers read. 

(PT - BR) Enunciado da Questão:

* Faça um programa que leia um número inteiro positivo N (máximo = 10) e depois N números inteiros e armazene-os em um vetor. Em seguida, mostrar na tela todos os números negativos lidos.
