(EN - US) Question Statement:

* Make a program to read a set of N names of students, as well as the grades they got in the 1st and 2nd semesters. Each of this information is to be stored in a vector. Then, print out the names of the approved students, considering approved those whose average grade is greater than or equal to 6.0 (six).

(PT - BR) Enunciado da Questão:

* Fazer um programa para ler um conjunto de N nomes de alunos, bem como as notas que eles tiraram no 1º e 2º semestres. Cada uma dessas informações deve ser armazenada em um vetor. Depois, imprimir os nomes dos alunos aprovados, considerando aprovados aqueles cuja média das notas seja maior ou igual a 6.0 (seis).
