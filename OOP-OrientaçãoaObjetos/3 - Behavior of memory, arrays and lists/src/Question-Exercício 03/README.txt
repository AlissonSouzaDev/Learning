(EN - US) Question Statement:

* Make a program to read the name, age and height of N people, as per example. Then, show on the the average height of these people, and also show the percentage of people under 16, as well as the names of these people if there are any. 

(PT - BR) Enunciado da Questão:

* Fazer um programa para ler nome, idade e altura de N pessoas, conforme exemplo. Depois, mostrar na tela a altura média das pessoas, e mostrar também a porcentagem de pessoas com menos de 16 anos, bem como os nomes dessas pessoas caso houver.
