(EN - US) Question Statement:

* You have a data set containing the height and gender (M, F) of N people. Make a program that calculates and writes the highest and lowest height of the group, the average height of the women, and the number of men.

(PT - BR) Enunciado da Questão:

* Tem-se um conjunto de dados contendo a altura e o gênero (M, F) de N pessoas. Fazer um programa que calcule e escreva a maior e a menor altura do grupo, a média de altura das mulheres, e o número de homens.
