(EN - US) Example Statement:

* Make a program to read an integer N and the height of N people. Store the N heights in an array. Then show the average height of these people.

(PT - BR) Enunciado da Exemplo:

* Fazer um programa para ler um número inteiro N e a altura de N pessoas. Armazene as N alturas em um vetor. Em seguida, mostrar a altura média dessas pessoas.
