(EN - US) Example Statement:

* Make a program to read an integer N and the data (name and price) of N products. Store the N products in an array. Then show the average price of the products.

(PT - BR) Enunciado do Exemplo:

* Fazer um programa para ler um número inteiro N e os dados (nome e preço) de N produtos. Armazene os N produtos em um vetor. Em seguida, mostrar o preço médio dos produtos.
