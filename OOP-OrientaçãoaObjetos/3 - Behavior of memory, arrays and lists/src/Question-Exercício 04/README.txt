(EN - US) Question Statement:

* Make a program that reads N integers and stores them in an array. Then show on the screen all the even numbers, and also the number of even numbers. 

(PT - BR) Enunciado da Questão:

* Faça um programa que leia N números inteiros e armazene-os em um vetor. Em seguida, mostre na tela todos os números pares, e também a quantidade de números pares.
