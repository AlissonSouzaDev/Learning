(EN - US) Question Statement:

* Make a program to read a vector of N integers. Then, show on the screen the arithmetic of only the read even numbers, with one decimal place. If no even numbers are entered, display the message "NO EVEN NUMBER".

(PT - BR) Enunciado da Questão:

* Fazer um programa para ler um vetor de N números inteiros. Em seguida, mostrar na tela a média aritmética somente dos números pares lidos, com uma casa decimal. Se nenhum número par for digitado, mostrar a mensagem "NENHUM NUMERO PAR".
