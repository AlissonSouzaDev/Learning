(EN - US) Question Statement:

* Make a program to read an integer N and then a vector of N real numbers. Then, display on the screen the arithmetic mean of all elements to three decimal places. Then show all elements of the vector that are below the average, to one decimal place each.

(PT - BR) Enunciado da Questão:

* Fazer um programa para ler um número inteiro N e depois um vetor de N números reais. Em seguida, mostrar na tela a média aritmética de todos elementos com três casas decimais. Depois mostrar todos os elementos do vetor que estejam abaixo da média, com uma casa decimal cada.
