(EN - US) Question Statement:

* Make a program to read two integers M and N, and then read an M row by N column matrix containing integers, and there may be repetitions. Then read an integer X that belongs to the matrix. For each occurrence of X, show the values to the left, above, right, and below X.

(PT - BR) Enunciado da Questão:

* Fazer um programa para ler dois números inteiros M e N, e depois ler uma matriz de M linhas por N colunas contendo números inteiros, podendo haver repetições. Em seguida, ler um número inteiro X que pertence à matriz. Para cada ocorrência de X, mostrar os valores à esquerda, acima, à direita e abaixo de X.
